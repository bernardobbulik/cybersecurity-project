Sobre o grupo

Este projeto foi desenvolvido por um grupo formado por três estudantes do 1º semestre do curso de Ciência da Computação da Universidade de Passo Fundo. A equipe reuniu seus conhecimentos iniciais em desenvolvimento web, adquiridos na disciplina Web Lab, utilizando HTML, CSS e JavaScript para criar um site informativo, acessível e útil para conscientizar os usuários sobre boas práticas de proteção digital e demonstrar domínio nas tecnologias citadas.
